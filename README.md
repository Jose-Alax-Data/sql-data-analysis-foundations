# sql-data-analysis-foundations
Projeto de fundamentos em análise de dados com SQL, focado em consultas básicas, validação de dados e métricas de negócio.  Inclui: - Limpeza e validação de dados - Análise de clientes e produtos - Métricas de vendas (soma, média, máximos e mínimos)  Objetivo: desenvolver raciocínio analítico e boas práticas em SQL.

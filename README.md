# SQL Data Analysis Foundations

## Descrição
Projeto focado em fundamentos de análise de dados com SQL, abordando consultas básicas, validação de dados e métricas de negócio.

---

## Objetivo
Desenvolver raciocínio analítico e aplicar boas práticas em SQL para análise de dados.

---

## Tecnologias Utilizadas
- SQL  
- MySQL  

---

## Análises Realizadas
- Validação e limpeza de dados  
- Identificação de inconsistências em emails  
- Análise de clientes e produtos  
- Cálculo de métricas de vendas (soma, média, máximo e mínimo)  

---

## Estrutura do Projeto
- `sql_analysis.sql` → consultas organizadas por problema  

---

## Diferencial
O projeto foi estruturado de forma progressiva, iniciando com consultas básicas e evoluindo para análises mais consolidadas, demonstrando evolução no raciocínio analítico.

---

## Observações
Este projeto utiliza validações simplificadas (como verificação de email via LIKE), com possibilidade de aprimoramento futuro utilizando expressões regulares (REGEXP).

---

## Próximos Passos

- Utilizar GROUP BY e HAVING para agregações mais avançadas  
- Integrar análises com ferramentas como Excel ou Power BI
